## 🔗 Healthcare Data Analysis Project  
## 🔗 Project Overview  

This project focuses on basic healthcare data analysis using Python. The main objective was to understand patient data and extract meaningful insights such as age distribution, treatment costs, and relationships between different health factors.
The project includes data cleaning, analysis, and visualization to present the findings in a clear and understandable way.

## 🔗 Key Analysis  
-Verified the dataset and confirmed there are no missing values  
-Calculated key metrics such as:  
Average age,
BMI,
Treatment cost.  
-Compared:
Diabetic vs non-diabetic patients,
Smokers vs non-smokers  
-Analyzed the correlation between blood pressure and cholesterol  
-Created multiple visualizations for better understanding of data  

## 🔗 Insights  
-The average age of patients is around 53 years  
-The dataset has an almost equal distribution of male and female patients  
-Approximately 47% of patients are diabetic  
-There is no strong correlation between blood pressure and cholesterol  
-Patients aged above 65 years tend to visit hospitals more frequently  
-Treatment costs vary significantly, with some patients having very high expenses  
-Smoking did not show a clear increase in treatment cost in this dataset  

## 🔗 Visualizations Included  
Histogram – Age distribution  
Bar Chart – Gender comparison  
Scatter Plot – BMI vs Treatment Cost  
Line Chart – Hospital visits by age group  
Pie Chart – Smoking distribution
